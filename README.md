# RAG-Chatbot
A simple chatbot that answers queries based on json data
---
**Requires ollama running on localhost:11434 with qwen3:4b-q4_K_M and nomic-embed-text models available**
# Instructions
1. Clone the repo<br>
`git clone https://github.com/sinkobobux/RAG-Chatbot.git && cd RAG-Chatbot`
2. Install the dependencies<br>
`pip install -r requirements.txt`
3. Run the app<br>
`python main.py`
