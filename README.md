# RAG-Chatbot
A simple chatbot that answers queries based on json data
---
**Requires Qwen3:4b and nomic-embed-text running on localhost:11434**
# Instructions
1. Clone the repo<br>
`git clone https://github.com/sinkobobux/RAG-Chatbot.git && cd RAG-Chatbot`
2. Install the dependencies<br>
`pip install -r requirements.txt`
3. Run the app<br>
`python main.py`
